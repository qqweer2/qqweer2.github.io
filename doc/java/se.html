环境配置

Windows

解压jdk文件包

通过 我的电脑 》 属性 》 高级系统设置 打开环境变量设置界面

配置 JAVA_HOME=jdk文件包路径

配置 CLASSPATH=.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar

window7 添加 PATH %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

window10添加 PATH %JAVA_HOME%\bin 和 %JAVA_HOME%\jre\bin 两条

Liunx

centOS7 vi /etc/pro.d/jdk.sh 创建文件配置如下


export JAVA_HOME=/opt/module/jdk1.8.0_341
export PATH=$PATH:$JAVA_HOME/bin
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar

执行 source /etc/profile 使环境生效

数字

格式化



        int number = 5;
        // 补足5位数字，不足部分前面补0
        String formatted = String.format("%05d", number);
        System.out.println(formatted); // 输出：00005

        // 如果数字超过5位，则不会添加前置0
        number = 123456;
        formatted = String.format("%05d", number);
        System.out.println(formatted); // 输出：12345

        // 如果需要左对齐，可以使用-标志
        formatted = String.format("%05d", number);
        System.out.println(formatted); // 输出：<12345  后面有空格

        // 如果需要0填充浮点数，可以使用如下方式
        double decimal = 123.45;
        formatted = String.format("%.1f", decimal);
        System.out.println(formatted); // 输出：123.5

        formatted = String.format("%.3f", decimal);
        System.out.println(formatted); // 输出：123.450

        // 使用0填充小数点后两位，不足位用0填充
        formatted = String.format("%05.2f", decimal);
        System.out.println(formatted); // 输出：0123.45

        // BigDecimal也支持，自动四舍五入
        formatted = String.format("%.1f", new BigDecimal("123.44"));
        System.out.println(formatted); // 输出：123.4
        formatted = String.format("%.1f", new BigDecimal("123.45"));
        System.out.println(formatted); // 输出：123.5
        // %d 抛出异常
        formatted = String.format("%.1d", new BigDecimal("123.45"));
        System.out.println(formatted); // 输出：123.5

        // 固定保留2位小数
        DecimalFormat format1 = new DecimalFormat("0.00");
        // 最多保留2位小数
        DecimalFormat format2 = new DecimalFormat("0.##");
        // 至少保留2位整数
        DecimalFormat format3 = new DecimalFormat("00");
        format1.setRoundingMode(RoundingMode.HALF_UP);
        format2.setRoundingMode(RoundingMode.HALF_UP);
        System.out.println(format1.format(new BigDecimal("3.40")));
        System.out.println(format2.format(new BigDecimal("3.40")));
        System.out.println(format3.format(120));
        System.out.println(format3.format(5));
