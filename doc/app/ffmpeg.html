命令

音频

转为MP3音频

ffmpeg -i 输入文件.m4a -codec:a libmp3lame 输出文件.mp3

截取音频

ffmpeg -i 输入文件.m4a -codec:a libmp3lame -ss 15 输出文件.mp3

提取MP3音频

ffmpeg -i 输入文件.mp4 -codec:a libmp3lame 输出文件.mp3

提取WAV音频

ffmpeg -i 输入文件.mp4 -codec:a pcm_s16le 输出文件.mp3

探测音频格式

>ffprobe -i 输入文件 -show_streams -show_format -print_format json
